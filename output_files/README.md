Pipeline outputs are limited due to size restrictions.

fastqc_data.txt = raw read quality check (Phred scores)
initial_busco_summary.txt = busco analysis after flye assembly
initial_quast_report.pdf = quast analysis after flye assembly
medaka.gbk = polished + assembled genome gene/protein annotations
medaka_vs_reference.mmap.normal.png = circos plot aligning polished + annotated genome to reference genome
polished_busco_summary.txt = busco analysis after medaka polishing
polished_quast_report.pdf = quast analysis after medaka polishing
reference.gbk = reference gene/protein annotations
variants_only.vcf = variant calling information (SNP, indels, ploidy)
